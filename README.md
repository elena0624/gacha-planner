# 課金規劃助手 🎮 (Gacha Planner)

這是一個專為課金規劃設計的可拖拉式網頁，讓您輕鬆安排活動期間（如 `Day 1 ~ Day N`）的課金項目與預算控制，並能自動與您的 Google 試算表（Google Sheets）進行雲端儲存與讀取同步。

## 🌟 升級版新功能
1. **多活動分頁儲存 (Multi-Event Sheets)**：可以在網頁設定「儲存分頁名稱」（例如：`Plan_Summer2026`、`Plan_Collab2026`），儲存時會在同一個試算表中**自動建立獨立的標籤分頁**，讓您輕鬆切換與保留不同活動的紀錄！
2. **網頁端品項編輯 (Item CRUD)**：可以直接在左側列表中「新增」、「編輯」或「刪除」品項，點選儲存後，修改過的品項列表會**自動寫回**試算表的 `GachaItems` 分頁中，再也不用手動去改 Excel 儲存格了！
3. **多人去中心化設計**：每個人只需使用這個網頁前端，設定自己部署的專屬 Google Apps Script URL，就能把資料存在各自的雲端硬碟，互不相干，完全保護個人穩私且 100% 免費。

## 🚀 串接 Google 試算表步驟（必看！）

為了讓網頁能直接讀寫您的 Google 試算表，請按照以下簡單步驟設定：

### 1. 貼上 Apps Script 腳本
1. 打開或新建一個 Google 試算表。
2. 點選上方選單的 **「擴充功能 (Extensions)」** -> **「Apps Script」**。
3. 刪除裡面原本預設的所有程式碼。
4. 打開專案底下的 `gas-code.js` 檔案，複製裡面所有的內容，貼到 Apps Script 編輯器中。
5. 點擊上方的 **「儲存 (Save)」** 按鈕（磁碟圖示）。

### 2. 部署為網頁應用程式 (Web App)
1. 點擊右上角的 **「部署 (Deploy)」** 鈕 -> **「新增部署 (New deployment)」**。
2. 點選左側的齒輪圖示，確保類型為 **「網頁應用程式 (Web app)」**。
3. 設定選項：
   - **說明 (Description)**: `課金規劃 API`
   - **專案執行身分 (Execute as)**: 選擇 **「我 (Me)」**。
   - **誰有權存取 (Who has access)**: 選擇 **「任何人 (Anyone)」**。 *(⚠️ 這非常重要，否則網頁會因為權限不足無法連線！)*
4. 點擊 **「部署」**。
5. 系統會跳出授權視窗，請點選您的 Google 帳號。
6. 如果顯示「Google 還未驗證此應用程式」的警告，請點選 **「進階 (Advanced)」** -> **「前往『未命名專案』（不安全）」**，然後點選 **「允許 (Allow)」**。
7. 部署成功後，複製畫面上顯示的 **「網頁應用程式 URL」**。

## 💻 網頁使用方式

1. 用任意瀏覽器直接雙擊打開 [index.html](file:///D:/side_project/gacha-planner/index.html)。
2. 將剛才複製的 **Google Apps Script URL** 貼到網頁上方的輸入框中，然後點擊 **「連線」**。
   - 連線成功後，系統會自動在 LocalStorage 記錄 URL，下次打開網頁就不用重新輸入。
   - 網頁會自動辨識您試算表中的品項欄位，並列出在左側「可課金品項」區。
   - 若是全新的試算表，系統會自動在雲端產生預設的品項（如月卡、禮包等）方便您測試。
3. **自訂活動與日期**：
   - 輸入您的活動開始與結束日期，設定「儲存分頁名稱」（例如：`Plan_Summer2026`），點選 **「生成日程」**。
4. **開始規劃**：
   - 從左側將品項**拖曳**到右側對應的 `Day X` 卡片中。
   - 每個天數的卡片會自動計算該日累計金額，網頁頂部會顯示「總花費」。
   - 點擊卡片品項右邊的 `X` 按鈕，可以移除已安排的項目。
5. **在網頁上管理品項**：
   - 滑鼠懸停在左側品項卡片上，會出現「鉛筆」與「垃圾桶」圖示，可以直接編輯或刪除該品項。
   - 點選左上角 **「新增品項」** 按鈕，可以建立全新的選購項目。
6. 點選 **「儲存至雲端」**，規劃結果便會自動回寫到您的試算表中（會在您的試算表內自動新增 `GachaItems` 存放品項，以及您設定的分頁名稱如 `Plan_Summer2026` 存放排程規劃）。
7. **切換不同活動計畫**：
   - 只需要在網頁的 **「儲存分頁名稱」** 欄位輸入您之前存過的其他計畫名稱（例如 `Plan_Collab2026`），再點選 **「讀取雲端」**，網頁就會自動載入並切換到該活動的日期與規劃！
